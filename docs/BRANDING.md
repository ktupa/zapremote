# ZAP Remote - Identidade Visual e Branding

Guia completo de identidade visual do ZAP Remote.

## 🎨 Conceito Visual

O ZAP Remote representa **tecnologia**, **confiança** e **conexão**. A identidade visual deve transmitir profissionalismo e modernidade.

## 🎯 Elementos Principais

### Nome
**ZAP Remote**
- Sempre com maiúsculas em "ZAP"
- "Remote" em maiúsculas apenas na inicial
- Sem hífen ou espaços extras

### Slogan
**"Acesso remoto seguro e confiável"**

Alternativas:
- "Conecte-se de qualquer lugar"- "Suporte remoto profissional"
- "Sua conexão segura"

## 🎨 Paleta de Cores

### Cores Principais

**Azul Tecnológico (Primary)**
- HEX: `#0066CC`
- RGB: `0, 102, 204`
- Uso: Logo, botões primários, destaques

**Verde Ativo (Secondary)**
- HEX: `#00CC66`
- RGB: `0, 204, 102`
- Uso: Status online, confirmações, ícones de sucesso

### Cores de Suporte

**Cinza Escuro**
- HEX: `#2C3E50`
- RGB: `44, 62, 80`
- Uso: Textos, backgrounds escuros

**Cinza Claro**
- HEX: `#ECF0F1`
- RGB: `236, 240, 241`
- Uso: Backgrounds claros, separadores

**Branco**
- HEX: `#FFFFFF`
- RGB: `255, 255, 255`
- Uso: Textos em fundos escuros

**Vermelho Alerta**
- HEX: `#E74C3C`
- RGB: `231, 76, 60`
- Uso: Erros, alertas críticos

## 🖥️ Aplicações Visuais

### Instalador Windows

**Imagens necessárias:**

1. **WizardImage** (164x314 pixels)
   - Lateral esquerda do instalador
   - Gradiente azul → branco
   - Logo ZAP Remote no topo

2. **WizardSmallImage** (55x58 pixels)
   - Canto superior direito
   - Logo simplificado
   - Fundo transparente

### Interface do Cliente

O RustDesk permite customização limitada no cliente. Foque no:
- Ícone da aplicação
- Nome exibido "ZAP Remote"
- Configuração pre-definida do servidor

## 💡 Dicas Importantes

1. **Simplicidade**: Logos simples funcionam melhor em tamanhos pequenos
2. **Escalabilidade**: Teste o logo em vários tamanhos
3. **Contraste**: Certifique-se de que funciona em fundos claros e escuros
4. **Legibilidade**: O nome deve ser legível mesmo em 16x16 pixels
5. **Consistência**: Use sempre as mesmas cores e proporções

## 🔄 Próximos Passos

3. Coloque em `/opt/zap-remote/branding/`
4. Atualize o script Inno Setup apontando para os arquivos
5. Recompile o instalador
